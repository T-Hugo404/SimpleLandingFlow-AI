# 8. Instruções para Criação de Sessões, CSS e JS

Prefixo padrão para IDs das sessões: _______

## 8.1 Identificação das Sessões (IDs)

1. Todas as sessões do projeto devem possuir um ID obrigatório.
2. O ID deve seguir o formato: [PREFIXO DEFINIDO] + [NÚMERO DA SESSÃO].
3. O prefixo deve ser o mesmo para todas as sessões do projeto.
4. Os números das sessões devem ser sequenciais conforme aparecem na página.

---

## 8.2 Responsividade

5. Todo o site deve ser completamente responsivo para desktop e mobile.
6. Sempre utilizar corretamente o sistema de grid, flex-box, containers e utilitários do Bootstrap 5.

---

## 8.3 Uso Prioritário do Bootstrap

7. O Bootstrap deve ser sempre a primeira opção de layout e estilização.
8. CSS ou JS personalizados só devem ser criados quando o Bootstrap não for suficiente.

---

## 8.4 Criação de CSS e JS Extras

9. Se o Bootstrap não for suficiente, aplicar a base Bootstrap na sessão e criar arquivos CSS e JS exclusivos para ela.
10. Esses arquivos extras devem complementar o Bootstrap, nunca substituí-lo.

---

## 8.5 Nomeação dos Arquivos CSS/JS

11. O nome dos arquivos CSS/JS deve ser exatamente o ID da sessão + extensão.
12. Esses arquivos devem ser incluídos apenas na página onde a sessão existe.
